ListAllEvent AllFlows (Called on any flow) /TenantName/Data's Flow/ListAllEvent FlowAdmin "Data's flow name"
Summary
SMART provides a set of events that can be used by all flows. These events are typically raised on the flow admin and provides helper services. The ListAllEvent is one such event that can be called on any flow to retrieve all the data for a given object. The event takes in a size parameter that limits the number of data objects retrieved.
Request Format
{
  "FlowAdmin": {
    "___smart_action___": "lookup",
    "___smart_value___": "Data's Flow name"
  },
  "group": "Object Name",
  "size": size to retrieve
}
Request Description
Name
Value
Description
group Object Name The name of the object to be retrived. The object has to be present in the flow for which this event is posted.
size Number The number of objects to be retrieved. This does not guarantee any order and just picks up the first size data objects and returns them. If the total number of objects present is less than size, all objects are returned.
Response Format
{
  "event": "Event id for the request",
  "responses": [
    {
      "resultSet": [
        list of data objects
      ],
      "___smart_responseid___": "response id"
    }
  ]
}
Response Description
Name
Value
Description
resultSet List of objects This returns a list of objects in the format of the data. The number of elements will be size or less than size.
