Flow
URI Format
Raised on Object
CreateUser Security /TenantName/Security/CreateUser FlowAdmin "Security"
Summary
SMART provides out of the box security that allows users to be associated to roles and access granted based on roles. SMART hence provides a user functionality that allows to create a single user and associate multiple identities to the same user. The user can login using any of the identities. The CreateUser event creates a new user in SMART for the given tenant. Call the AddIdentity event to create an identity for the user. Unless an identity is added no permissions can be used.
Request Format
{
  "FlowAdmin": {
    "___smart_action___": "lookup",
    "___smart_value___": "Security"
  },
  "id": "user name",
  "name": "Actual name of user",
  "roles": [
    list of roles to grant to user
  ]
}
Request Description
Name
Value
Description
id A user name An unique identifier identifying this user.
name The name of the user SMART only requires a user name to create a user and no other information.
roles list of roles A list of role names of the roles to be associated with this user. More than one role can be associated with the same user. The AddRoleToUser event can also be called to add roles to a user once the user is created.
Response Format
{
  "event": "Event id of the request",
  "responses": [

      "message": "Created a user for user name : actual name",
      "___smart_responseid___": "Response id"
    }
  ]
}
Response Description
Name
Value
Description
message Create user Indicates that the user has been created or an user with the id is already existing.
