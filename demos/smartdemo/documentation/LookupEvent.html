Event Name
Flow
URI Format
Raised on Object
LookupEvent AllFlows(Data Object Flow) /TenantName/Data's Flow/LookupEvent FlowAdming "Data's Flow name"
Summary
SMART provides a set of events that can be used by all flows. These events are typically raised on the flow admin and provides helper services. The LookupEvent is one such event that returns a single data object for the given key.
Request Format
{
  "FlowAdmin": {
    "___smart_action___": "lookup",
    "___smart_value___": "Object flow"
  }
  "group": "Object to lookup",
  "key": "Key of the object to lookup",
}
Request Description
Name
Value
Description
group Object name The name of the object to lookup.
key key value The value of the key of the object to lookup.
Response Format
{
  "event": "Event id for the request",
  "responses": [
    {
      "result": [
         List of looked up objects
      ],
      "___smart_responseid___": "Response id"
    }
  ]
}
Response Description
Name
Value
Description
result Data looked up The data that corresponds to the given key. The data is in the format of the Object that is looked up. The member variables of the object is the name and the value is the value of the variable.
