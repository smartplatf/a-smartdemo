URI Format
Raised on Object
ListDeployments AdminSmartFlow /SmartOwner/AdminSmartFlow/ListDeployments, /TenantName/AdminSmartFlow/ListDeployments TenantAdmin "SmartOwner"
Summary
The ListDeployments gives back the list of deployments based on the request. This event can be raised on either the SmartOwner tenant or any tenant. When raised on the SmartOwner tenant, this returns the list of deployed flows. When raised on a tenant, this lists enabled deployment types. If no flow is provided, this lists the enabled flows for the tenant. If a flow is provided, the dtype indicates the deployment type to retrieve. When it is event, it retrieves the events of the flow, when it is data it retrieves the data for the flow.
Request Format
{
  "FlowAdmin": {
    "___smart_action___": "lookup",
    "___smart_value___": "AdminSmartFlow"
  }
  "flow":"Flow name",
  "dtype":"event/data/links"
}
Request Description
Name
Value
Description
flow flow name This is an optional value. When provided this retrieves the deployments for the flow.
dtype deployment type This is an optional value. When provided it retrieves the deployments of the given type for the given flow.
Response Format
{
  "event": "Event id of the request",
  "responses": [
    {
      "enabledFlows": [
        List of enabled flows
      ],
      "___smart_responseid___": "Response id"
    }
  ]
}

{
  "event": "Event id of the request",
  "responses": [
    {
      "dType": "type of deployment"
      "deployments": [
        List of deployments
      ],
      "___smart_responseid___": "Response id"
    }
  ]
}
Response Description
Name
Value
Description
enabledFlows Flows enabled for tenant If the request is for enabled flows, then this is returned.

deployments List of Deployments If the request is for a deployment then the list is returned in this.
