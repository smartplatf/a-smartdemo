Raised on Object
GetCartItems CartFlow /TenantName/CartFlow/GetCartItems The Cart for which to retrieve items
Summary
This is a demo event for the Plug and Play demo. This event retrieves the related cart items for the Cart object. The implementation for this event shows how to pass as a parameter the list of linked objects for the cart object.
Request Format
{
  "Cart": {
    "___smart_action___": "lookup",
    "___smart_value___": "name of the cart"
  }
}
Request Description
Name
Value
Description
Response Format
{
  "event": "Event id for the request",
  "responses": [
    {
      "cartName": "Name of the cart",
      "totalItems": number of items in cart,
      "items": [
        List of items in cart
      ],
      "___smart_responseid___": "response id"
    }
  ]
}
Response Description
Name
Value
Description
cartName Cart Name The name of the cart for which data is retrieved
totalItems Number Number of items in cart
items List of cartItems This returns a list of CartItem objects that belong to this cart. This only returns the key of the related object. Call the LookupEvent to get the actual object.
