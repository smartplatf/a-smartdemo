Event Name
Flow
URI Format
Raised on Object
DeployEvent AdminSmartFlow /SmartOwner/AdminSmartFlow/DeployEvent TenantAdmin "SmartOwner"
Summary
SmartOwner is the default tenant created in every SMART platform and is used to administer SMART. The DeployEvent is used to deploy jars into SMART for usage by different tenants. Deploying a jar does not enable it for any tenant.
Request Format
{
  "TenantAdmin": {
    "___smart_action___": "lookup",
    "___smart_value___": "SmartOwner"
  },
  "deployJar": "file path on local filesystem",
  "flowsoa": "The soa file name"
}
Request Description
Name
Value
Description
deployJar Jar file The file name of the jar to be deployed. This file needs to be present in the local file system of the server.
flowsoa Flow soa file name This is the file that is present within the jar, describing the jar.
Response Format
{
  "event": "Event id of the request",
  "responses": [
    {
      "success": "Done File Name",
      "___smart_responseid___": "Response id"
    }
  ]
}
Response Description
Name
Value
Description
success Done message Indicates that this jar has been deployed
