Summary
The WriteReview is a demo event and part of the Plug and Play feature. This demonstrates how we can mash up an event to be raised on an object which is not a part of this flow and not tightly coupled. The object on which the event should be raised is specified when the flow is enabled for a tenant.
Request Format
{
  "Mashed object name": {
    "___smart_action___": "lookup",
    "___smart_value___": "mashed object key value"
  }
  "reviewText": "The review text",
  "rating": "A rating for the product",
  "reviewObject": "The skuId or the mashed up object's link",
}
