Event Name
Flow
URI Format
Raised on Object
AddToCart CartFlow /TenantName/CartFlow/AddToCart Cart to which to add item
Summary
This is a demo event that is part of the Plug and Play Demo. This event is in CartFlow and adds an item that is mashed up with this flow to the cart. The key passed for the item is looked up for validity before it is linked. If not present, this will throw and exception.
Request Format
{
  "Cart": {
    "___smart_action___": "lookup",
    "___smart_value___": "Cart Name"
  }
  "cartItem": "Key of the item that is linked",
}
Request Description
Name
Value
Description
cartItem Key of item to be added This should be a valid key and should be saved for the given type mashed up with the CartFlow.
Response Format
{
  "event": "Event id for the request",
  "responses": [
    {
      "message": "Added item to cart: cart Name",
      "___smart_responseid___": "Response id"
    }
  ]
}
Response Description
Name
Value
Description
message Added item Indicates that the item has been added.
