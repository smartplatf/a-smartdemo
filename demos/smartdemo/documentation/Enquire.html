Event Name
Flow
URI Format
Raised on Object
Enquire ContactFlow /TenantName/ContactFlow/Enquire Contact "Name of contact to send enquiry"
Summary
This is a demo event that is used to showcase executing an event without a authentication. An enquiry can be done by any user without a valid user name/password in the system. This event has been setup to run for an unauthenticated access.
Request Format
{
  "subject": "A subject to send in the enquiry",
  "message": "A message to include in the enquiry",
  "name": "Name of the person",
  "email": "email of the person",
  "phone": "phone of the person",
  "Contact": {
    "___smart_action___": "lookup",
    "___smart_value___": "contact to send enquiry to"
  }
}
Request Description
Name
Value
Description
subject subject A subject for the enquiry.
message A message A message for the enquiry.
name name Name of the enquirer.
email email Email of the enquirer.
phone phone Phone of the enquirer.
Response Format
{
  "event": "Event id of request",
  "responses": [
    {
      "message": "An email has been sent.",
      "___smart_responseid___": "Response id"
    }
  ]
}
Response Description
Name
Value
Description
message An email has been sent. While the demo does not send an email, this can be integrated to the out of box service provided by SMART to send out emails to the contact.
