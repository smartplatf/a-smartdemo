CloseEnquiry ContactFlow /TenantName/ContactFlow/CloseEnquiry Enquiry "id of the enquiry to be closed"
Summary
This is a demo event to showcase access of an event only based on role. The event can only be raised by a user with the sales role. The event just changes the state of the enquiry from active to inactive, so that the enquiry does not appear again.
