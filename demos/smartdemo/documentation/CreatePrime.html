CreatePrime AllFlows (Any Flow) /TenantName/Data's Flow/CreatePrime FlowAdmin "Flow name of the data created"
Summary
SMART provides a set of events that can be used by all flows. These events are typically raised on the flow admin and provides helper services. The CreatePrime is one such service that creates Prime Data of any flow. This calls the standard transitions that are written for the event. Customizations can be added to this by providing a transition for the FlowAdmin and CreatePrime combination with the name of the object to be created.
Request Format
{
  "FlowAdmin": {
    "___smart_action___": "lookup",
    "___smart_value___": "Flow to which object belongs"
  }
  "create": "Object name to be created",
  "data": "{ Data to create object }",
}
Request Description
Name
Value
Description
create Object Name Specify the name of the data object to be created here. This should belong to the flow on which this event is called.
data The data to be used to create object Provide a map of the data to be used to create the object. This is a name-value pair, where the name matches the member variable name in the object and the value type matches the type of the member variable.
