DeActivateMessage MessageFlow /TenantName/MessageFlow/DeActivateMessage Message "The message to be deactivated"
Summary
This is a demo event that showcases the custom event functionality in SMART. This event just changes the state of a Message from active to inactive so that the message is not displayed any more. The inactive state is created as an end state. Once a data goes into an end state none of the events can be raised on it.
